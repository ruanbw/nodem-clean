## [1.1.0] - 2026-05-10

### Added
- CLI 顶层错误兜底：命令执行失败时输出友好错误并返回非 0 退出码。
- 扫描流程加入有限并发（默认 8），提升大目录扫描性能。
- README 新增行为说明：默认跳过符号链接、禁止根目录扫描。

### Changed
- `dirSize` 从递归实现改为迭代遍历，降低极深目录的栈深风险。
- 路径处理统一为规范化绝对路径，并校验目标必须是目录。

### Fixed
- 扫描时 `readdir/lstat` 异常容错增强，避免单点异常导致整体中断。
- 删除阶段增强容错：单个目录删除失败不会中断后续删除，并输出失败项。
- 修正目录大小显示单位：由字节误标为 `kb` 改为正确换算后的 `KB`。
- 未找到 `node_modules` 时给出明确提示并结束流程。

### Security / Safety
- 拦截根目录（如 `/`、`C:\`）扫描，降低误操作风险。
- 默认不跟随符号链接，避免跨目录误删。
